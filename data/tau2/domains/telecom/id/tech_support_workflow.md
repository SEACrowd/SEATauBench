# Perangkat Ponsel - Alur Kerja Pemecahan Masalah Dukungan Teknis

## Pendahuluan

Dokumen ini memberikan alur kerja terstruktur untuk mendiagnosis dan menyelesaikan masalah teknis ponsel. Ikuti jalur ini berdasarkan deskripsi masalah user. Setiap langkah mencakup panduan tentang tindakan pemecahan masalah spesifik mana yang harus dilakukan berdasarkan apa yang perlu diperiksa atau dimodifikasi.

Pastikan Anda mencoba semua langkah penyelesaian yang relevan sebelum mentransfer user ke agen manusia.

## Referensi Tindakan Pengguna yang Tersedia
Berikut adalah tindakan yang dapat dilakukan oleh user pada perangkat mereka.
Anda harus memahami tindakan tersebut dengan baik karena sebagai bagian dari dukungan teknis Anda harus membantu pelanggan melakukan serangkaian tindakan

Catatan: tindakan diagnostik di bawah ini adalah tindakan yang dilakukan user di ponsel mereka, bukan tool yang dapat Anda panggil langsung. Minta user melakukannya dan laporkan hasilnya.

Agen harus memandu pengguna untuk melakukan tindakan spesifik ini sesuai kebutuhan selama pemecahan masalah:


### Tindakan Diagnostik (Hanya-baca)
1. **Periksa Bilah Status** - Menampilkan ikon apa saja yang saat ini terlihat di bilah status ponsel Anda (area di bagian atas layar). Menampilkan kekuatan sinyal jaringan, status data seluler (diaktifkan, dinonaktifkan, penghemat data), status Wi-Fi, dan tingkat baterai.
2. **Periksa Status Jaringan** - Memeriksa status koneksi ponsel Anda ke jaringan seluler dan Wi-Fi. Menampilkan status mode pesawat, kekuatan sinyal, jenis jaringan, apakah data seluler diaktifkan, dan apakah roaming data diaktifkan. Kekuatan sinyal bisa berupa "tidak ada", "lemah" (1 batang), "cukup" (2 batang), "baik" (3 batang), "sangat baik" (4+ batang).
3. **Periksa Preferensi Mode Jaringan** - Memeriksa preferensi mode jaringan ponsel Anda. Menampilkan jenis jaringan seluler yang lebih disukai oleh ponsel Anda untuk disambungkan (misalnya, 5G, 4G, 3G, 2G).
4. **Periksa Status SIM** - Memeriksa apakah kartu SIM Anda berfungsi dengan benar dan menampilkan statusnya saat ini. Menampilkan apakah SIM sedang aktif, hilang, atau terkunci dengan kode PIN atau PUK.
5. **Periksa Pembatasan Data** - Memeriksa apakah ponsel Anda memiliki fitur pembatas data aktif. Menampilkan apakah mode Penghemat Data aktif dan apakah penggunaan data latar belakang dibatasi secara global.
6. **Periksa Pengaturan APN** - Memeriksa pengaturan APN teknis yang digunakan ponsel Anda untuk terhubung ke jaringan data seluler operator Anda. Menampilkan nama APN saat ini dan URL MMSC untuk pengiriman pesan gambar.
7. **Periksa Status Wi-Fi** - Memeriksa status koneksi Wi-Fi Anda. Menampilkan apakah Wi-Fi diaktifkan, jaringan mana yang sedang terhubung (jika ada), dan kekuatan sinyalnya.
8. **Periksa Status Panggilan Wi-Fi** - Memeriksa apakah Panggilan Wi-Fi diaktifkan di perangkat Anda. Fitur ini memungkinkan Anda untuk melakukan dan menerima panggilan melalui jaringan Wi-Fi, bukan menggunakan jaringan seluler.
9. **Periksa Status VPN** - Memeriksa apakah Anda menggunakan koneksi VPN (Virtual Private Network). Menampilkan apakah VPN sedang aktif, terhubung, dan menampilkan detail koneksi available.
10. **Periksa Aplikasi Terinstal** - Mengembalikan nama semua aplikasi yang terinstal di ponsel.
11. **Periksa Status Aplikasi** - Memeriksa informasi terperinci tentang aplikasi tertentu. Menampilkan izin dan pengaturan penggunaan data latar belakangnya.
12. **Periksa Izin Aplikasi** - Memeriksa izin apa saja yang saat ini dimiliki aplikasi tertentu. Menampilkan apakah aplikasi memiliki akses ke fitur seperti penyimpanan, kamera, lokasi, dll.
13. **Jalankan Tes Kecepatan** - Mengukur kecepatan koneksi internet Anda saat ini (kecepatan unduh). Memberikan informasi tentang kualitas koneksi dan aktivitas apa saja yang dapat didukungnya. Kecepatan unduh bisa berupa "tidak diketahui", "sangat lemah", "lemah", "cukup", "baik", atau "sangat baik".
14. **Dapat Mengirim MMS** - Memeriksa apakah aplikasi perpesanan dapat mengirim pesan MMS.

### Tindakan Perbaikan (Tulis/Modifikasi)
1. **Setel Mode Jaringan** - Mengubah jenis jaringan seluler yang lebih disukai oleh ponsel Anda untuk disambungkan (misalnya, 5G, 4G, 3G). Jaringan berkecepatan tinggi (5G, 4G) memberikan data lebih cepat tetapi mungkin lebih menguras baterai.
2. **Alihkan Mode Pesawat** - Mengaktifkan atau menonaktifkan Mode Pesawat. Saat AKTIF, fitur ini memutuskan semua komunikasi nirkabel termasuk seluler, Wi-Fi, dan Bluetooth.
3. **Pasang Ulang Kartu SIM** - Mensimulasikan pelepasan dan pemasangan kembali kartu SIM Anda. Ini dapat membantu mengatasi masalah pengenalan.
4. **Alihkan Data Seluler** - Mengaktifkan atau menonaktifkan koneksi data seluler ponsel Anda. Mengontrol apakah ponsel Anda dapat menggunakan data seluler untuk akses internet saat Wi-Fi tidak tersedia.
5. **Alihkan Roaming Data** - Mengaktifkan atau menonaktifkan Roaming Data. Saat AKTIF, roaming diaktifkan dan ponsel Anda dapat menggunakan jaringan data di area di luar cakupan operator Anda.
6. **Alihkan Penghemat Data** - Mengaktifkan atau menonaktifkan mode Penghemat Data. Saat AKTIF, fitur ini mengurangi penggunaan data, yang mungkin memengaruhi kecepatan data.
7. **Setel Pengaturan APN** - Mengatur pengaturan APN untuk ponsel.
8. **Atur Ulang Pengaturan APN** - Mengatur ulang pengaturan APN Anda ke pengaturan default.
9. **Alihkan Wi-Fi** - Mengaktifkan atau menonaktifkan radio Wi-Fi ponsel Anda. Mengontrol apakah ponsel Anda dapat menemukan dan terhubung ke jaringan nirkabel untuk akses internet.
10. **Alihkan Panggilan Wi-Fi** - Mengaktifkan atau menonaktifkan Panggilan Wi-Fi. Fitur ini memungkinkan Anda untuk melakukan dan menerima panggilan melalui Wi-Fi, bukan menggunakan jaringan seluler, yang dapat membantu di area dengan sinyal seluler yang lemah.
11. **Hubungkan VPN** - Terhubung ke VPN (Virtual Private Network) Anda.
12. **Putuskan Koneksi VPN** - Memutuskan koneksi VPN (Virtual Private Network) aktif apa pun. Menghentikan perutean lalu lintas internet Anda melalui server VPN, yang mungkin memengaruhi kecepatan koneksi atau akses ke konten.
13. **Berikan Izin Aplikasi** - Memberikan izin tertentu ke aplikasi (seperti akses ke penyimpanan, kamera, atau lokasi). Diperlukan agar beberapa fungsi aplikasi berfungsi dengan baik.
14. **Mulai Ulang Perangkat** - Memulai ulang ponsel Anda sepenuhnya. Ini dapat membantu mengatasi banyak gangguan perangkat lunak sementara dengan menyegarkan semua layanan dan koneksi yang berjalan.

## Klasifikasi Masalah Awal

Tentukan kategori mana yang paling menggambarkan masalah user:

1. **tidak ada layanan/Masalah Koneksi**: Ponsel menampilkan "tidak ada layanan" atau tidak dapat terhubung ke jaringan
2. **Masalah Data Seluler**: Tidak dapat mengakses internet atau mengalami kecepatan data lambat
3. **Masalah Pesan Gambar/Grup (MMS)**: Tidak dapat mengirim atau menerima pesan gambar

Untuk beberapa masalah, atasi konektivitas dasar terlebih dahulu.

## Jalur 1: tidak ada layanan / Pemecahan Masalah Tidak Ada Koneksi

### Langkah 1.0: Periksa apakah user menghadapi masalah tidak ada layanan
Jika layanan available, bilah status tidak akan menampilkan 'tidak ada sinyal' atau 'mode pesawat'.
- Minta user untuk memeriksa bilah status mereka
- Jika bilah status menunjukkan bahwa layanan available, user tidak menghadapi masalah tidak ada layanan.
- Jika bilah status menunjukkan bahwa layanan bukan available, lanjutkan ke Langkah 1.1

### Langkah 1.1: Periksa Mode Pesawat dan Status Jaringan
Minta user untuk memeriksa koneksi ponsel mereka ke jaringan seluler dan Wi-Fi. Ini akan menunjukkan apakah Mode Pesawat aktif, kekuatan sinyal, dan detail koneksi lainnya.

**Jika Mode Pesawat AKTIF:**
- Minta user untuk menonaktifkan Mode Pesawat
- Minta user untuk melihat bilah status mereka dan memeriksa apakah layanan telah dipulihkan

**Jika Mode Pesawat DINONAKTIFKAN:**
- Lanjutkan ke Langkah 1.2

### Langkah 1.2: Verifikasi Status Kartu SIM
Minta user untuk memeriksa apakah kartu SIM mereka berfungsi dengan benar. Anda perlu tahu apakah SIM tersebut hilang, terkunci, atau aktif.

**Jika SIM ditampilkan sebagai HILANG:**
- Minta user untuk memasang ulang kartu SIM dengan melepas dan memasangnya kembali
- Periksa apakah kartu SIM AKTIF.
- Minta user untuk melihat bilah status mereka dan memeriksa apakah layanan telah dipulihkan

**Jika SIM TERKUNCI dengan PIN/PUK:**
- Lakukan eskalasi ke dukungan teknis untuk bantuan mengenai keamanan SIM

**Jika SIM AKTIF dan berfungsi:**
- Lanjutkan ke Langkah 1.3

### Langkah 1.3: Coba atur ulang pengaturan APN
Jika masalah konektivitas dasar berlanjut:

- Minta user untuk mengatur ulang pengaturan APN ke default
- Minta mereka untuk memulai ulang perangkat mereka
- Minta user untuk melihat bilah status mereka dan memeriksa apakah layanan telah dipulihkan

**Jika masih belum teratasi:**
- Lanjutkan ke Langkah 1.4

### Langkah 1.4: Periksa Penangguhan Saluran

Tidak ada layanan bisa disebabkan oleh saluran yang ditangguhkan.

**Jika saluran ditangguhkan:**
- Ikuti instruksi dalam kebijakan umum untuk informasi lebih lanjut tentang penangguhan saluran dan cara mencabut penangguhan.
- Jika Anda dapat mencabut penangguhan:
    - Minta user untuk melihat bilah status mereka dan memeriksa apakah layanan telah dipulihkan.
- Jika Anda tidak dapat mencabut penangguhan:
    - Lakukan eskalasi ke dukungan teknis.

**Jika masih belum teratasi:**
- Lakukan eskalasi ke dukungan teknis

## Jalur 2: Pemecahan Masalah Data Seluler Tidak Tersedia atau Lambat

Catatan: Jalur ini tidak mencakup masalah data wifi.

### Langkah 2.0: Periksa apakah user menghadapi masalah data

Saat data seluler tidak tersedia, tes kecepatan harus mengembalikan 'tidak ada koneksi'.
Jika data available, tes kecepatan juga akan mengembalikan kecepatan data. Kecepatan apa pun di bawah 'Sangat baik' dianggap lambat.
- Jalur 2.1 periksa masalah data seluler tidak tersedia.
- Jalur 2.2 periksa masalah data lambat.

## Jalur 2.1: Pemecahan Masalah Data Seluler Tidak Tersedia

### Langkah 2.1.0: Periksa apakah user menghadapi masalah data seluler tidak tersedia

- Minta user untuk menjalankan tes kecepatan.
- Jika tes kecepatan mengembalikan 'tidak ada koneksi', data seluler tidak tersedia.
    - Ikuti Jalur 2.1.
    - Setelah masalah teratasi, jika kecepatan tidak 'Sangat baik', ikuti Jalur 2.2.
- Jika tes kecepatan mengembalikan kecepatan data, data seluler available.
    - Jika kecepatan 'Sangat baik', user tidak menghadapi masalah data seluler.
    - Untuk kecepatan lainnya apa pun ('Lemah', 'Cukup', 'Baik'), data seluler mungkin lambat dan Anda harus mengikuti Jalur 2.2.

### Langkah 2.1.1: Verifikasi Masalah Layanan
Minta user untuk memeriksa apakah ponsel mereka memiliki layanan seluler. Data seluler memerlukan setidaknya beberapa koneksi jaringan seluler.

- Ikuti langkah pemecahan masalah Jalur 1 (tidak ada layanan / Tidak Ada Koneksi) terlebih dahulu.
- Setelah Anda mengonfirmasi bahwa layanan available, periksa apakah masalah data seluler berlanjut.
    - Minta user untuk menjalankan ulang tes kecepatan dan memeriksa konektivitas data.
    - Jika masih tidak ada konektivitas, lanjutkan ke Langkah 2.1.2.

### Langkah 2.1.2: Verifikasi apakah user sedang bepergian
Minta user jika mereka berada di luar area layanan biasa mereka.

**Jika Pengguna tidak bepergian:**
- Lanjutkan ke Langkah 2.1.3

**Jika Pengguna bepergian:**
- Minta user untuk memverifikasi apakah Roaming Data diaktifkan untuk memungkinkan penggunaan data pada jaringan lainnya.

**Jika Roaming Data DINONAKTIFKAN:**
- Minta user untuk mengaktifkan Roaming Data
- Minta mereka untuk menjalankan ulang tes kecepatan dan memeriksa konektivitas data.

**Jika Roaming Data AKTIF tetapi tidak berfungsi:**
- Verifikasi bahwa saluran yang terkait dengan nomor ponsel yang diberikan oleh user diaktifkan untuk roaming.
    - Jika saluran tidak diaktifkan untuk roaming, aktifkan tanpa biaya untuk user
- Minta user untuk menjalankan ulang tes kecepatan dan memeriksa konektivitas data.
    - Jika masih tidak ada konektivitas, lanjutkan ke Langkah 2.1.3.

**Jika Roaming Data AKTIF dan diaktifkan tetapi konektivitas tidak berfungsi:**
- Lanjutkan ke Langkah 2.1.3

### Langkah 2.1.3: Periksa Pengaturan Data Seluler
**Jika Data Seluler DINONAKTIFKAN:**
- Minta user untuk mengaktifkan Data Seluler
- Minta user untuk menjalankan ulang tes kecepatan dan memeriksa konektivitas data.
    - Jika masih tidak ada konektivitas, lanjutkan ke Langkah 2.1.4.

**Jika Data Seluler AKTIF tetapi tidak berfungsi:**
- Lanjutkan ke Langkah 2.1.4

### Langkah 2.1.4: Periksa Penggunaan Data
Periksa apakah, untuk saluran yang terkait dengan nomor ponsel yang diberikan oleh user, penggunaan data user telah melebihi batas data mereka.

**Jika Penggunaan Data DILAMPAUI:**
- Minta user apakah mereka ingin mengubah paket lain atau mengisi ulang data.
- Ikuti instruksi dalam kebijakan umum untuk informasi lebih lanjut tentang pengisian ulang data dan perubahan paket.
- Jika Anda dapat mengisi ulang data atau mengubah ke paket dengan batas data yang lebih tinggi:
    - Minta user untuk menjalankan ulang tes kecepatan dan memeriksa konektivitas data.
    - Jika masih tidak ada konektivitas, transfer ke dukungan teknis.
- Jika Anda tidak dapat mengisi ulang data atau mengubah ke paket dengan batas data yang lebih tinggi (tidak diizinkan atau user tidak menginginkannya):
    - Lakukan eskalasi ke dukungan teknis.

**Jika Penggunaan Data TIDAK DILAMPAUI:**
- Minta user untuk menjalankan tes kecepatan dan memeriksa konektivitas data.
    - Jika masih tidak ada konektivitas, transfer ke dukungan teknis.

## Jalur 2.2: Pemecahan Masalah Data Seluler Lambat

### Langkah 2.2.0: Periksa apakah user menghadapi masalah data lambat
Saat data seluler available tetapi kecepatan apa pun lainnya dari 'Sangat baik', user menghadapi masalah data lambat.
- Minta user untuk menjalankan tes kecepatan.
- Jika tes kecepatan mengembalikan 'tidak ada koneksi', data seluler tidak tersedia.
    - Ikuti Jalur 2.1.
- Jika tes kecepatan mengembalikan kecepatan data, data seluler available.
    - Jika kecepatan 'Sangat baik', user tidak menghadapi masalah data lambat.
    - Untuk kecepatan lainnya apa pun ('Lemah', 'Cukup', 'Baik'), data seluler mungkin lambat dan Anda harus mengikuti Jalur 2.2.

### Langkah 2.2.1: Periksa Pengaturan Pembatasan Data
Minta user untuk memeriksa apakah ada pengaturan yang membatasi penggunaan data mereka, seperti mode Penghemat Data.

**Jika Penghemat Data AKTIF:**
- Minta user untuk menonaktifkan mode Penghemat Data
- Minta user untuk menjalankan ulang tes kecepatan dan memeriksa apakah kecepatan meningkat menjadi 'Sangat baik'.
    - Jika bukan ini masalahnya, lanjutkan ke Langkah 6.
**Jika Penghemat Data DINONAKTIFKAN:**
- Lanjutkan ke Langkah 6

### Langkah 2.2.2: Periksa Preferensi Mode Jaringan
Minta user untuk memeriksa jenis jaringan seluler apa yang lebih disukai oleh ponsel mereka. Menggunakan mode lama seperti 2G/3G dapat membatasi kecepatan secara signifikan.

**Jika diatur ke jenis jaringan lama (hanya 2G/3G):**
- Minta user untuk mengubah preferensi jaringan ke opsi yang menyertakan 5G
- Minta user untuk menjalankan ulang tes kecepatan dan memeriksa apakah kecepatan meningkat menjadi 'Sangat baik'.
    - Jika bukan ini masalahnya, lanjutkan ke Langkah 7.

**Jika sudah pada pengaturan optimal:**
- Lanjutkan ke Langkah 7

### Langkah 2.2.3: Periksa VPN Aktif
Minta user untuk memeriksa apakah mereka menggunakan VPN (Virtual Private Network) yang mungkin memengaruhi kualitas koneksi.

**Jika VPN aktif:**
- Minta user untuk menonaktifkan koneksi VPN mereka saat ini
- Minta mereka untuk menjalankan ulang tes kecepatan dan memeriksa apakah kecepatan meningkat menjadi 'Sangat baik'.
    - Jika bukan ini masalahnya, lakukan eskalasi ke dukungan teknis.

**Jika tidak ada VPN atau menonaktifkan tidak membantu:**
- Lakukan eskalasi ke dukungan teknis.

## Jalur 3: Pemecahan Masalah MMS (Pesan Gambar/Grup)

### Langkah 3.0: Periksa apakah user menghadapi masalah MMS
Saat MMS tidak berfungsi, user tidak akan dapat mengirim atau menerima pesan gambar.

- Minta user jika mereka dapat mengirim pesan MMS menggunakan aplikasi perpesanan default.
    - Jika ini berfungsi, user tidak menghadapi masalah MMS.
    - Jika ini tidak berfungsi, lanjutkan ke Langkah 3.1.

### Langkah 3.1: Verifikasi Status Layanan Jaringan
Minta user untuk memeriksa apakah ponsel mereka memiliki layanan seluler. MMS memerlukan setidaknya beberapa koneksi jaringan seluler.

- Ikuti langkah pemecahan masalah Jalur 1 (tidak ada layanan / Tidak Ada Koneksi) terlebih dahulu.
- Setelah Anda mengonfirmasi bahwa layanan available, periksa apakah masalah berlanjut:
    - Minta user jika mereka dapat mengirim pesan MMS menggunakan aplikasi perpesanan default.

**Jika layanan available:**
- Lanjutkan ke Langkah 3.2

### Langkah 3.2: Verifikasi Status Data Seluler
Data seluler diperlukan untuk MMS.

- Gunakan langkah pemecahan masalah Jalur 2.1 (Data Seluler Tidak Tersedia) untuk memeriksa apakah konektivitas data seluler berfungsi. Jangan khawatir tentang kecepatan, fokuslah pada konektivitas.
- Setelah Anda mengonfirmasi bahwa konektivitas data seluler berfungsi, periksa apakah masalah MMS berlanjut:
    - Minta user untuk mencoba mengirim pesan MMS menggunakan aplikasi perpesanan default lagi.

### Langkah 3.3: Periksa Teknologi Jaringan
Minta user untuk memeriksa jenis jaringan seluler apa yang ponsel mereka terhubung ke. MMS memerlukan setidaknya teknologi 3G atau lebih tinggi.

**Jika terhubung ke jaringan 2G saja:**
- Minta user untuk mengubah mode jaringan untuk menyertakan setidaknya 3G/4G/5G
- Minta user untuk mencoba mengirim pesan MMS menggunakan aplikasi perpesanan default lagi.

**Jika pada jaringan 3G atau lebih tinggi:**
- Lanjutkan ke Langkah 3.4


### Langkah 3.4: Periksa Status Panggilan Wi-Fi
Minta user untuk memeriksa apakah Panggilan Wi-Fi diaktifkan, karena dapat mengganggu fungsionalitas MMS.

**Jika Panggilan Wi-Fi AKTIF:**
- Minta user untuk menonaktifkan Panggilan Wi-Fi
- Minta user untuk mencoba mengirim pesan MMS menggunakan aplikasi perpesanan default lagi.

**Jika Panggilan Wi-Fi DINONAKTIFKAN atau menonaktifkannya tidak membantu:**
- Lanjutkan ke Langkah 3.5

### Langkah 3.5: Verifikasi Izin Aplikasi Perpesanan
Minta user untuk memeriksa bahwa aplikasi perpesanan default memiliki izin yang diperlukan - khususnya izin penyimpanan dan SMS.

**Jika izin penyimpanan atau SMS hilang:**
- Minta user untuk memberikan kedua izin yang diperlukan ke aplikasi perpesanan
- Minta user untuk mencoba mengirim pesan MMS menggunakan aplikasi perpesanan default lagi.

**Jika semua izin diberikan:**
- Lanjutkan ke Langkah 3.6

### Langkah 3.6: Periksa Pengaturan APN
Minta user untuk memeriksa pengaturan teknis (APN) yang digunakan ponsel mereka untuk terhubung ke jaringan data seluler operator.

**Secara khusus periksa:**
- Konfigurasi URL MMSC (harus ada agar MMS berfungsi)

**Jika URL MMSC hilang:**
- Minta user untuk mengatur ulang pengaturan APN ke default operator
- Minta user untuk mencoba mengirim pesan MMS menggunakan aplikasi perpesanan default lagi.

**Jika masalah berlanjut setelah memeriksa semua di atas:**
- Lakukan eskalasi ke dukungan teknis
